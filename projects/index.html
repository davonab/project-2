Java-ATM Machine

IIT Bank and Trust Company wants a program that will resemble a typical ATM machine. This program will allow their customers to enter a pin and if it is correct, a new window will open showing options such as Savings Account, Checking Account, and Loan Payment. For the savings account option, users can either add or withdraw money from their account. For loan payments, users have options to pay for their mortgage, student loans, personal loans, or auto loans. For checking account, a user can simply write a check to take out a certain amount of money. After every input, a final balance will be displayed to the user.

C++ TV Remote

Create a program that represents a TV remote. The output to the user will show the current channel and sound volume. You will display options that will allow the user to turn the TV off, select a different channel, go up a channel, go down a channel, select a new sound volume, go up a volume level, go down a volume level, adda new channnel, remove a channel, and display all channels.
